School project FYS

Deze web applicatie moet niet gebruikt worden een echte website, het is namelijk een school project. De naam van het project is Fasten Your Seatbelt, in het kort FYS. Het project wordt door scholieren van de HVA in hun eerste jaar software development gemaakt. Deze specifieke applicatie is gemaakt door: Joey, Badr, Jimi en Kerim.

Wie zijn wij?

Wij zijn een groep van 4 studenten die werken onder de naam PubG. Onze namen zijn Badr, Jimi Joey en Kerim. Wij studeren op de HVA Hbo ICT Software engineering. Deze website is een schoolproject dat wij moesten maken met samenwerking van Corendon. Het doel van dit project is om een site te maken voor volwassene die geen reispartner hebben. Voor ons allemaal was dit iets nieuws om aan zo een groot project te werken. We hebbenn hier veel van geleerd dat websites maken ook buiten coderen veel werk is.
